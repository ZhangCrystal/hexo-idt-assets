# hexo-asset-images


Give asset image in hexo a absolutely path automatically

# Usege

```shell
npm install hexo-idt-assets --save
```

# Example

```shell
hello
├── hello.jpg
└── text.txt

hello.md
```

Make sure `post_asset_folder: true` in your `_config.yml`.

Just use `[介绍](path/hello.jpg)` to insert `hello.jpg`.
